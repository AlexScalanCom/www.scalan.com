Disable corporate computer blocking

11 Sep 2023 AlexS
arduino

Many corporate environments enforce automatic screen locking after a period of inactivity — often just 10 minutes. While this is a reasonable security policy, it can be frustrating when you're reading documentation, watching a long-running process, or simply stepping away briefly and don't want your session to lock.

The solution: a tiny Arduino-based device that periodically simulates subtle mouse movement, keeping the system alive without anyone noticing.

How It Works

The device uses an Arduino Leonardo (or any ATmega32U4-based board) connected via USB. Unlike standard Arduinos, the Leonardo natively supports USB HID emulation, meaning it can act as a real mouse or keyboard.

The firmware moves the cursor by just 2 pixels in a random direction every 60–120 seconds — enough to prevent idle lockout, but too subtle to be distracting on screen. An onboard LED blinks once per movement cycle so you can confirm it is running.

The Firmware

Upload the following sketch to your Arduino Leonardo via the Arduino IDE. Make sure you have the correct board selected (Tools > Board > Arduino Leonardo).

#include <Mouse.h>

void setup() {
  randomSeed(analogRead(0));
  Mouse.begin();
  pinMode(LED_BUILTIN, OUTPUT);
}

void loop() {
  delay(60000 + random(60000)); // 60 seconds + random(0..60) seconds
  switch (random(4)) { // random direction
    case 0:
      Mouse.move(2, 0, 0);
      break;
    case 1:
      Mouse.move(-2, 0, 0);
      break;
    case 2:
      Mouse.move(0, 2, 0);
      break;
    case 3:
      Mouse.move(0, -2, 0);
      break;
  }
  // Led indicator blink.
  digitalWrite(LED_BUILTIN, HIGH);  // turn the LED on
  delay(1000);                      // wait for a second
  digitalWrite(LED_BUILTIN, LOW);   // turn the LED off
}

Code Breakdown

SnippetPurpose
randomSeed(analogRead(0))Seeds the random generator from an unconnected analog pin for true randomness
Mouse.begin()Initializes USB HID mouse emulation
delay(60000 + random(60000))Waits 1–2 minutes between movements
Mouse.move(...)Shifts cursor by 2 pixels in a randomly chosen direction
digitalWrite(LED_BUILTIN, ...)Blinks the built-in LED as a running indicator

Usage

  1. Flash the firmware to the Arduino Leonardo using the Arduino IDE.
  2. Plug the board into any available USB port on your computer.
  3. The device starts automatically — no drivers or software installation required.
  4. Watch for a subtle LED blink every 1–2 minutes to confirm it is working.

Customization

  • Adjust interval — change 60000 + random(60000) to suit your corporate policy. For a 5-minute lockout, 300000 + random(60000) (5–6 minutes) is safe.
  • Change movement distance — modify the 2 in Mouse.move() for larger or smaller cursor shifts.
  • Add keyboard emulation — include <Keyboard.h> and send harmless key events if your system also locks on pure mouse inactivity.

A cheap, self-contained hardware solution that keeps your session alive with zero footprint on the host system.
