Classical Tetris Game

14 Oct 2022 AlexS
game javascript

Tetris is one of those timeless games that never gets old. First released in 1984, the core mechanic remains deceptively simple: rotating and positioning falling blocks to clear complete horizontal lines. Simple enough for anyone to pick up, yet deep enough to challenge even veteran players.

This project is a modern JavaScript implementation of the classic Tetris game that runs entirely in your browser — no installs, no plugins.

How to Play

Spin up a game session and use your keyboard to control the falling pieces:

KeyAction
↑ (Up Arrow)Rotate piece clockwise
← (Left Arrow)Move piece left
→ (Right Arrow)Move piece right
↓ (Down Arrow)Soft drop — accelerate descent
SpaceHard drop — instant placement

Game Features

  • Classic 7-bag randomizer — all seven tetromino types (I, O, T, S, Z, J, L) appear in balanced rotation
  • Score tracking — earn more points for clearing multiple lines at once
  • Increasing difficulty — pieces fall faster as you progress
  • Responsive controls — snappy input with no lag

Try It

Ready to play? Open the game in a new tab and see how high you can score.
