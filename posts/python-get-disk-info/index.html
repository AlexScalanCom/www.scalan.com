Python Get Disk Free Space and Render It on an Image

15 Feb 2021 AlexS
python linux raspberryPi

Running out of disk space on a Raspberry Pi silently breaks logging, database writes, and cron jobs. Displaying free disk space directly on your dashboard image gives you instant storage awareness without opening an SSH terminal. The psutil library provides filesystem stats, and Pillow renders the value as rotated text baked into your canvas.

This guide shows how to fetch free disk space in MB and draw it vertically on an existing image.

Why Display Disk Space on Screen?

  • Early warning — notice low storage before services start failing
  • Kiosk-friendly — full-screen dashboards don't have a terminal window for df -h
  • Single-glance metrics — combine disk info with CPU temp, uptime, and network stats on one screen
  • No cron alerts needed — visual indicator that updates with every refresh cycle

The Solution

import psutil
from PIL import Image, ImageDraw, ImageFont

# Get disk usage for root partition
hdd = psutil.disk_usage('/')

# Calculate free space in MB
free_mb = int(hdd.free / (1024.0 ** 2))
text = f"Free: {free_mb} MB"

# Load font and render text to grayscale image
font = ImageFont.truetype('./freefont/FreeMonoBold.ttf', 22)
img_txt = Image.new('L', font.getsize(text), 255)
draw_txt = ImageDraw.Draw(img_txt)
draw_txt.text((0, 0), text, font=font, fill=0)

# Rotate 90° for vertical sidebar display and paste onto canvas
t = img_txt.rotate(90, expand=1)
image.paste(t, (155, 5))

Required Dependencies

Install the required packages:

pip3 install psutil Pillow

Make sure you have a TrueType font file (e.g., FreeMonoBold.ttf) in your project directory.

Understanding the Code

ComponentPurpose
psutil.disk_usage('/')Returns a named tuple with total, used, free, and percent for the root partition
hdd.free / (1024.0 ** 2)Converts bytes to megabytes (divide by 1024²)

ImageFont.truetype()Loads a TrueType font at the specified size (22pt)
font.getsize(text)Returns (width, height) tuple for canvas sizing and positioning
Image.new('L', ...)Creates an 8-bit grayscale image sized exactly to fit the text
ImageDraw.Draw()Provides drawing methods for rendering text onto the image

How It Works

  1. Query filesystem — psutil.disk_usage('/') reads partition statistics from the OS
  2. Convert to MB — divides free bytes by 1024² and rounds down to an integer
  3. Render text — creates a small grayscale image sized to the string, draws black text on white background
  4. Rotate 90° — turns horizontal label into vertical text for sidebar placement
  5. Paste onto canvas — places the rotated disk info label at (155, 5) on the main dashboard image

Practical Examples

Show Usage Percentage Instead of MB

hdd = psutil.disk_usage('/')
text = f"Used: {hdd.percent}%"

Color-Code by Free Space Threshold

Change text color based on remaining storage:

free_mb = int(hdd.free / (1024.0 ** 2))
if free_mb > 5000:
    color = (0, 255, 0)     # green - plenty of space
elif free_mb > 1000:
    color = (255, 255, 0)   # yellow - getting low
else:
    color = (255, 0, 0)     # red - critical

Monitor a Specific Partition

Check a mounted data volume instead of root:

hdd = psutil.disk_usage('/mnt/data')
text = f"/mnt/data: {int(hdd.free / (1024.0**2))} MB free"

Show Total, Used and Free Together

hdd = psutil.disk_usage('/')
total_mb = int(hdd.total / (1024.0 ** 2))
used_mb = int(hdd.used / (1024.0 ** 2))
free_mb = int(hdd.free / (1024.0 ** 2))
text = f"{used_mb}/{total_mb} MB ({hdd.percent}%)"

Customization Tips

  • Modern font sizing — replace font.getsize() with font.getbbox(text) for Pillow ≥10 compatibility (getsize is deprecated)
  • All partitions — use psutil.disk_partitions(all=True) to list every mounted filesystem
  • Multiple disk labels — paste separate rotated images at different Y coordinates for multi-disk systems
  • Live refresh loop — wrap in a while True with time.sleep(60) for updates every minute

One filesystem query, one image render. Free space visible at a glance, storage crises avoided before they start.
