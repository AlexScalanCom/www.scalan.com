Linux Mouse Mover Bash Script to Prevent Screensaver

26 Jul 2023 AlexS
bash linux DevOps

Desktop environments lock screens or dim displays after periods of inactivity. When you're running long compilations, watching logs, or monitoring a remote session without touching the keyboard, an idle screen blocker can be a serious annoyance. A simple bash script that injects tiny mouse movements keeps your session alive without any visible cursor jitter.

This guide shows how to build a mouse mover script using evemu-event to simulate 1-pixel movements at random intervals.

Why Automate Mouse Movement?

  • Prevent screen lock — keep the session active during long-running terminal tasks
  • Block power save — stop your monitor from entering sleep or dim mode
  • Zero visual distraction — 1-pixel moves are imperceptible to anyone looking at the screen
  • No third-party daemons — a plain bash script with one CLI dependency does the job

The Solution

Save the following script as keep-mouse-activity.sh:

#!/bin/bash

#
# Emulating movement of mouse (randomly move up or
# down or left or right to 1 pixel every 10 seconds).
#

_dev=$(ls -1 /dev/input/by-id/*event-mouse | tail -1)
[ -z "${_dev}" ] && echo "Cannot find mouse event device" && exit 1

echo "Using ${_dev}"

while [ true ]
do
        sleep 10
        case $(($RANDOM%4)) in
          0)
            echo "Up"
            /usr/bin/evemu-event ${_dev} --type EV_REL --code REL_Y --value -1 --sync
            ;;
          1)
            echo "Down"
            /usr/bin/evemu-event ${_dev} --type EV_REL --code REL_Y --value 1 --sync
            ;;
          2)
            echo "Left"
            /usr/bin/evemu-event ${_dev} --type EV_REL --code REL_X --value -1 --sync
            ;;
          3)
            echo "Right"
            /usr/bin/evemu-event ${_dev} --type EV_REL --code REL_X --value 1 --sync
            ;;
        esac
done

Make it executable and run:

chmod +x keep-mouse-activity.sh
./keep-mouse-activity.sh

Understanding the Script

ComponentPurpose
ls -1 /dev/input/by-id/*event-mouseLists all mouse input devices (one per line)
| tail -1Picks the last detected mouse device as the target
[ -z "${_dev}" ] && exit 1Guards against missing devices — exits with error if none found
sleep 10Waits 10 seconds between movements (adjust for different intervals)
$(($RANDOM%4))Generates a random number 0–3 to pick one of four directions
evemu-eventInjects synthetic input events into the kernel evdev subsystem
--type EV_REL --code REL_X/YSpecifies relative axis movement (horizontal or vertical)
--value ±1Moves exactly 1 pixel in the chosen direction
--syncFlushes the event to the device immediately

Prerequisite: Install evemu-tools if not already present: sudo apt install evemu-tools (Debian/Ubuntu) or sudo dnf install evemu (Fedora).

How It Works

  1. Device detection — scans /dev/input/by-id/ for mouse event devices and selects the last match
  2. Safety check — verifies a device was found; exits with an error message if not
  3. Infinite loop — while true keeps the script running until manually stopped (Ctrl+C)
  4. Random injection — every 10 seconds, picks a random direction and injects a 1-pixel movement via evemu-event

Practical Examples

Run in Background

nohup ./keep-mouse-activity.sh > /dev/null 2>&1 &
echo $!

The PID is printed so you can stop it later with kill <PID>.

Adjust Movement Interval

Change sleep 10 to sleep 30 for movements every 30 seconds, or sleep 5 for more frequent activity.

Stop the Script

Find and kill the running process:

pkill -f keep-mouse-activity.sh

Customization Tips

  • Larger movement — change --value ±1 to --value ±3 for more pronounced cursor shifts (still subtle)
  • Add diagonal moves — extend the case block with 8 directions by using $(($RANDOM%8)) and combining X + Y events
  • Random interval — replace sleep 10 with sleep $(($RANDOM % 20 + 5)) for unpredictable timing (5–25 seconds)
  • Systemd service — wrap the script in a systemd unit to auto-start on boot and manage via systemctl

A bash script that keeps your screen alive with imperceptible mouse nudges. Run it, walk away, come back to an unlocked desktop.
