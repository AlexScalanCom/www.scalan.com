Two-Player Tetris

22 Oct 2022 AlexS
game javascript

Tetris is legendary as a single-player experience, but what makes it even more fun is competition. This project brings the classic block-stacking puzzle to a two-player, split-screen format — both players share one keyboard and one screen, racing to clear lines before the other.

Built entirely with vanilla JavaScript, it runs right in your browser with zero setup.

Controls

Each player has a dedicated set of keys so both games run simultaneously without interference:

Left Player (WASD)

KeyAction
WRotate piece clockwise
AMove piece left
DMove piece right
SHard drop — instant placement
QSoft drop — accelerate descent

Right Player (Arrows)

KeyAction
↑ (Up Arrow)Rotate piece clockwise
← (Left Arrow)Move piece left
→ (Right Arrow)Move piece right
SpaceHard drop — instant placement
↓ (Down Arrow)Soft drop — accelerate descent

Game Features

  • Split-screen gameplay — two independent boards, one screen
  • Dual keyboard mapping — WASD for Player 1, Arrow keys for Player 2
  • Real-time scoring — track who clears lines faster
  • Progressive difficulty — pieces speed up as the game progresses
  • Instant restart — no page reload needed between rounds

Grab a friend (or challenge yourself with two hands) and dive in.
