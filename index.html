About

I'm a software engineer specializing in Java backend development, cloud infrastructure, and test automation. I enjoy building reliable software, automating development workflows, and improving the quality of applications through comprehensive testing.

Over the years, I've worked with modern Java technologies, AWS cloud services, CI/CD pipelines, relational databases, and frontend automation tools. My focus is on writing clean, maintainable code and delivering stable production-ready systems.

Java Backend Development

I build RESTful applications using Spring Boot with a strong emphasis on clean architecture and maintainability.

Typical responsibilities include:

  • Designing REST APIs.
  • Developing layered applications using Controllers, Services, and Repositories.
  • Implementing complex entity relationships with Hibernate.
  • Managing database schema changes using Liquibase.
  • Writing optimized SQL queries.
  • Configuring Maven and Gradle builds.
  • Integrating third-party services.

Testing & Quality Assurance

Reliable software starts with comprehensive automated testing.

My testing experience includes:

  • Unit testing with JUnit 5 and Mockito.
  • Integration testing using Spring Boot Test, H2, and MockMvc.
  • REST API functional and load testing with Apache JMeter.
  • End-to-end browser automation using Selenium WebDriver.
  • Building reusable automated test suites for authentication, user management, products, teams, and other business processes.

DevOps & Cloud

I have practical experience deploying and maintaining Java applications in AWS environments.

Typical tasks include:

  • Provisioning development and staging environments.
  • Configuring EC2, S3, ALB, Auto Scaling Groups, and RDS.
  • Building CI/CD pipelines with Jenkins.
  • Automating deployments to development, staging, and production.
  • Managing Linux servers and deployment scripts.
  • Supporting release and delivery processes.
