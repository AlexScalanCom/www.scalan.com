Develop Java Set of Services (Onboarding department)

  • 01 Mar 2025
fintech java

Project duration: 2 years · Domain: Fintech / Commercial Banking

Overview

Spearheaded the full lifecycle development of a modular microservices ecosystem deployed on the production infrastructure of a commercial banking platform. Each service was engineered for high availability, fault tolerance, and seamless integration with legacy Oracle and NoSQL (MongoDB) data stores — ensuring reliable real-time data processing at scale.

Responsibilities & Achievements

  • Cross-functional collaboration — Partnered closely with business analysts to translate complex domain requirements into clear, actionable technical specifications; facilitated sprint planning and refined user stories for optimal delivery.

  • Iterative development & testing — Prepared and validated feature increments through rigorous code reviews, local integration tests, and staging deployments before any production touchpoint.

  • Comprehensive test coverage — Authored unit tests (JUnit 5), integration tests (TestContainers, dbUnit), and behavior-driven tests (Cucumber/Gherkin) to guarantee data integrity and regression safety across all service boundaries.

  • Dev environment ownership — Orchestrated automated deployments to DEV environments using CI/CD pipelines, ensuring zero-downtime rollouts and rapid rollback capabilities.

  • Production-grade monitoring — Implemented continuous health checks, structured logging (ELK stack), and custom dashboards to monitor DEV server performance, quickly triaging anomalies before escalation.
