Support of WP Site on AWS

  • 24 Nov 2022
highload DevOps

Project duration: 4 years · Domain: High-Load Couriers Web Platform / AWS DevOps

Overview

Led the complete infrastructure build and long-term DevOps support of a high-traffic WordPress site hosted on AWS. Designed and provisioned multi-environment Elastic Beanstalk platforms (Dev, Test, Prod), orchestrated Jenkins-based CI/CD pipelines, and maintained 24/7 operational reliability through proactive monitoring and rapid incident response over a four-year engagement.

Responsibilities & Achievements

  • AWS infrastructure provisioning — Architected and deployed production-grade Elastic Beanstalk environments spanning EC2 compute fleets, Application Load Balancers (ALB), Auto Scaling Groups (ASG), elastic file storage via EFS, and managed MySQL databases through Amazon RDS — replicated across Dev, Test, and Prod stages.

  • Jenkins CI/CD server setup — Provisioned and hardened dedicated EC2 instances to host the Jenkins CI/CD control plane; configured secure access, plugin ecosystem, and credential management for seamless team onboarding.

  • WordPress deployment pipelines — Designed and maintained parameterized Jenkins pipelines to automate WordPress build, artifact packaging, and zero-downtime deployments across all three environments with gated promotions and rollback support.

  • Node.js microservice infrastructure — Standalone EC2-backed service instances running custom Node.js data-processing applications integrated with RDS MySQL; built dedicated CI/CD pipelines to deploy and version these services alongside the core WordPress platform.

  • Health monitoring & alerting — Developed automated Jenkins jobs performing scheduled health checks against all application endpoints and underlying EC2 instances; integrated CloudWatch alarms for real-time notification on CPU, memory, disk, and HTTP response anomalies.

  • Team support & knowledge transfer — Served as the primary DevOps point of contact for the entire engineering team; triaged production incidents, authored runbooks, and mentored junior engineers on AWS best practices and safe deployment procedures.

Delivered a resilient, auto-scaling AWS architecture that sustained consistent high-load traffic over four years with 99.9%+ uptime.
